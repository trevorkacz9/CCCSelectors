CSS Selectors and Specificity

Selector Name Explanation Example
. Class To select elements with a specific class .header {
element Element Selector Selects HTML elements based on the element name p {
element, element Group Selector Selects all HTML elements with the same style definitions h1, h2, p {
# id Selector Uses the id attritube of an HTML element to select a specific element #header
* Universal Selector Selects all HTML elements on the page * {

More Complex Examples

Selector(s) Example Description
.class1 .class2 .header .caption Selects all the elements with both the class "header" and "caption"
element.class p.header Selects all <p> elements with the class "header"
element element p h2 Selects all the <p> elements within the <h2> element
element>element div > p Selects all of the <p> elements where the parent element is <div>
